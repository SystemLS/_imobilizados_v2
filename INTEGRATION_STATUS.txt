│ ProcessExportJob:                                                               │
│   • Processa Excel/PDF em background                                           │
│   • Notifica via webhook ao completar                                          │
│   • Armazena em storage/exports/                                               │
│   • Suporta retry automático                                                   │

│                                                                                 │

│                                                                                 │
│ Comandos:                                                                       │
│   php artisan queue:work           → Iniciar worker                            │
│   php artisan queue:monitor        → Monitorar filas                           │
│   php artisan queue:failed         → Ver jobs falhados                         │
│   php artisan queue:retry all      → Reprocessar jobs                          │
